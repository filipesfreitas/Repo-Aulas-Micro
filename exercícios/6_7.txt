7. (a) Escreva uma função em C que calcula a função exponencial da seguinte forma:
	
Considere o cálculo até o termo n = 20. O protótipo da função é double ExpTaylor(double x);
    double ExpTaylor(double x){

    double exp;

    int i, n, fat;

    exp = 0;

    for(i = 0; i <= 20; i++){

        fat = 1;

        for(n = i; n > 1; n--){
            fat *= n;
        }

        exp += pow(x, i)/fat;

    }

    return exp;
}
