Detailed Description

CAPT_CommConfig.h This file configures the Captivate COMM stack.

Version
1.04.00.00 Released on Mon, Aug 15, 2016 5:06:13 PM

This is the Hhigh-level communications interface= layer. These functions provide the interfaces between the application level, protocol layer, and the low-level serial interface.
